My Projects

Over Which

Overwhich is a web application created to help players create the most effective team in Blizzard's popular game Overwatch. Players can easily visualize how different characters compliment each other while contributing to the database and community to make recommendations more accurate. Built with an Angular front-end on an express server framework. Check it out.

Rent-All

Spawned out of a barside conversation about how quickly property values were changing throughout Chicago, Rent-All is an app that makes it easy to explore how the housing market has changed in a given zip code. Select any US state and browse zip codes or enter what you're interested in directly to see how statistics like average price per square foot and average home price have fluctuated. The app uses Zillow's API and D3 for visualization.

School Patron

Ever wanted to help your local schools but been unsure where to start? School Patron is an app designed to help schools communicate what they need and donors to display what they have to offer. Built on an express backend with a Handlebars templating engine, School Patron is designed to make it easy to be a good samaritan.

Door Quest

A mash-up of a classic Choose-Your-Own-Adventure game with turn-based RPG elements, Door Quest is a spooky Lovecraftian adventure (and my very first web development project.) Tread carefully as you check it out, surviving the Old One's cultists isn't supposed to be easy!
