My Work

Overwhich

What is it? Overwhich is a web application created to help players create the most effective team in Blizzard's popular game Overwatch.

Why is it cool? Overwhich uses a unique database that responds to community feedback to help user's make intelligent decisions in a complex game. The application is flexible enough to make intelligent recommendations even as the 'meta' of the game changes and different playstyles become popular.

What's it built with? Overwhich is an Angular 4 front-end paired with an express server and a MongoDB database. Users can either use the front-end application or use the database as an Overwatch character API.


Rent-All

What is it? Spawned out of a barside conversation about how quickly property values were changing throughout Chicago, Rent-All is an app that makes it easy to explore how the housing market has changed in a given zip code. Select any US state and browse zip codes or enter what you're interested in directly to see how statistics like average price per square foot and average home price have fluctuated.

Why is it cool? The application uses D3 to draw detailed maps of any US state to show you exactly where the zip codes lie in a given state. If nothing else, you can marvel at how many zip codes the US has.

What's it built with? Rent-All is built on an Angular 4 front-end and a Sinatra server that uses a SQL database. Property data is provided by the Zillow API.


School Patron

What is it? School Patron is an app designed to make it easier for local schools to get the supplies they need! At it's core School Patron is a social network that allows schools to easily post what they need and for potential donors to post what they have to give. In short, School Patron aims to make it easier to be a good samaritan.

Why is it cool? School Patron utilizes a non-relational database that allows schools to easily list items they need, remove them when they're resolved, and give credit to the donor that helped them. The app utilizes many features that are often taken for granted - such as a search engine, user authentication, and the ability to send private messages.

What's it built with? School Patron is built with an express server and the handlebars templating engine.
